## 🚀 如何使用

### 立即验证（本地）

1. **启动开发服务器**
```bash
npm run dev
```

2. **访问 SEO 检查工具**
```
http://localhost:3000/seo-check.html
```

3. **查看结果**
- 绿色 ✅ = 通过
- 黄色 ⚠️ = 警告
- 红色 ❌ = 失败

### 部署后验证

1. **提交到搜索引擎**
   - Google Search Console
   - Bing Webmaster Tools
   - 百度站长平台

2. **测试社交分享**
   - Facebook Sharing Debugger
